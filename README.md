# Mydget

Web application to track transactions and maintain budget

## Developing

1. Clone repository

```bash
git clone https://github.com/PodYapolskiy/mydget.git
```

2. Install dependencies

```bash
npm install
```

3. Husky set up

```bash
npm run prepare
```

4. Running

```bash
npm run dev
```

## Building

To create a production version of your app:

```bash
npm run build
```
